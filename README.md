# 📊 Análisis de Comportamiento de Clientes: ConnectaTel Latam

Este proyecto realiza un análisis integral de datos para una empresa de telecomunicaciones ficticia en Latinoamérica. El objetivo es evaluar patrones de consumo, limpiar inconsistencias en la data y proponer estrategias de retención basadas en segmentación.

## 📁 Estructura del Proyecto
* `notebooks/`: Contiene el análisis detallado en Python (.ipynb).
* `datasets/`: Información de planes, usuarios y consumo histórico.

## 🛠️ Tecnologías Utilizadas
* **Python 3**
* **Pandas**: Limpieza y manipulación de datos.
* **Matplotlib & Seaborn**: Visualización estadística (Histogramas, Boxplots, Countplots).
* **Numpy**: Manejo de valores nulos y cálculos lógicos.

## 📈 Hallazgos Principales
* **Segmentación:** El 50% de la base de usuarios son adultos entre 30 y 60 años.
* **Uso:** El patrón de consumo más frecuente es el "Uso Medio", con una distribución sesgada a la derecha en el consumo de minutos.
* **Calidad de Datos:** Se realizó una limpieza profunda de valores "sentinel" (-999) y fechas futuras (2026).

## 🚀 Cómo ejecutar
1. Clona este repositorio.
2. Abre el archivo `.ipynb` en Google Colab o Jupyter Notebook.
3. Asegúrate de cargar los archivos CSV en la ruta especificada.
